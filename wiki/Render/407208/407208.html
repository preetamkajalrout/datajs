Project Description
datajs is a new cross-browser JavaScript library that enables data-centric web applications by leveraging modern protocols such as JSON and OData and HTML5-enabled browser features. It's designed to be small, fast and easy to use.

More information on how to use this library is available on the documentation page.

To discuss problems or provide feedback, please use the discussions page.

You can download the latest version from this site, or use the NuGet datajs package.
