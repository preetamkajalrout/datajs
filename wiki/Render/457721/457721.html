To get started using datajs, you can download the latest release datajs version 1.0.1. The Overview topic should get you started on how to use the library.

If you are using Visual Studio, you can also download the NuGet package.

There are two files available in datajs, where n.n.n are the major, minor and revision numbers.
  1. datajs-n.n.n.js. This file provides readable sources with comments and is the recommended file for development.
  2. datajs-n.n.n-min.js. This file is a minified version of the regular file and is the recommended file for deployment.
